# EasyAI – An Erasmus Project on AI Education & Sustainability

## Vision
The goal of EasyAI is to educate and raise awareness about Artificial Intelligence among students in Germany, Italy, Spain, and Turkey. We aim to highlight both the possibilities of AI and its specific risks, while strongly connecting the technology to the topic of sustainability.

## Run the project locally with Node.js
1. Install the latest **Node.js LTS** from [nodejs.org](https://nodejs.org/).
2. Open **PowerShell** or **Command Prompt** on Windows, or **Terminal** on macOS/Linux.
3. Go to the project folder:
   ```powershell
   cd C:\path\to\consume-minigame
   ```
4. Install the dependencies:
   ```powershell
   npm install
   ```
5. Start the local development server:
   ```powershell
   npm run dev
   ```
6. Open the local URL shown in the terminal, usually `http://localhost:5173`.

To create a production build, run:
```powershell
npm run build
```
